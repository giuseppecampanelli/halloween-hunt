# Halloween Hunt

COMP 376 Assignment 2

Giuseppe Campanelli 40043909

Duck Hunt with a Halloween spin.

## How To Run
1. Open Project in Unity
2. Open Menu Scene
3. Go to Game tab
4. Make sure Aspect Ratio is set to 'Standalone (1024x576)'
5. Click Play button
